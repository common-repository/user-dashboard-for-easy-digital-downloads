== Description ==
User Dashboard for Easy Digital Downloads is the best and only solution to create a clean, user-friendly, mobile-friendly dashboard for your customers.
Customers can easily change their account details. Also customers can view purchased items and download history. The plugin is in intitial stage so few things may be left.
No worries about dashboard anymore. Because this plugin will let you create a dashboard in few clicks.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `user-dashboard-for-easy-digital-downloads` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
